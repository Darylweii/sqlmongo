# AI Data Router Agent - 30道测试题目

## 测试覆盖范围

- **基础查询**（5题）：简单的单设备数据查询
- **设备搜索**（3题）：搜索和查找设备
- **项目查询**（2题）：项目列表和统计
- **对比查询**（4题）：多设备数据对比
- **统计分析**（3题）：数据统计和分析
- **复杂查询**（3题）：多条件复杂查询
- **时间范围查询**（3题）：不同时间范围的查询
- **聚合分析**（3题）：数据聚合和汇总
- **趋势分析**（2题）：数据趋势判断
- **异常检测**（2题）：异常数据识别

---

## 基础查询（1-5）

### 1. 简单 - 今日数据查询
**问题**：查询 a1_b9 设备今天的电量数据  
**预期**：返回 a1_b9 设备今天的 ep（电量）数据  
**测试点**：基础查询、时间解析（今天）

### 2. 简单 - 历史月份查询
**问题**：a1_b9 设备 2024年1月的电量  
**预期**：返回 a1_b9 设备 2024年1月的 ep 数据  
**测试点**：基础查询、月份范围解析

### 3. 简单 - 昨日数据查询
**问题**：查询 b1_b14 设备昨天的电流数据  
**预期**：返回 b1_b14 设备昨天的 i（电流）数据  
**测试点**：基础查询、时间解析（昨天）、不同数据类型

### 4. 中等 - 最近N天查询
**问题**：a1_b9 设备最近7天的用电量趋势  
**预期**：返回 a1_b9 设备最近7天的数据，包含趋势分析  
**测试点**：时间范围计算、趋势分析

### 5. 中等 - 日期范围查询
**问题**：查询 a1_b9 在 2024年1月5日到1月10日的电量数据  
**预期**：返回指定日期范围的数据  
**测试点**：精确日期范围解析
