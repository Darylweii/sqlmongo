# 更新日志

## [1.0.2] - 2026-02-03

### 修复
- ✅ **修复前端分页重复请求问题**
  - 问题：用户快速点击翻页按钮时，前端发送重复请求导致显示重复数据
  - 原因：前端没有防止并发请求的机制
  - 解决：添加 `isLoadingTable` 锁机制，确保同一时间只有一个请求
  - 效果：完全消除重复请求，用户体验显著提升

### 优化
- 🔒 **前端并发控制**
  - 添加请求锁机制（`isLoadingTable` 标志）
  - 在请求进行中禁用分页按钮
  - 使用 `finally` 确保锁一定被释放
  - 添加详细的控制台日志，方便调试

### 新增
- 🧪 **测试工具**
  - `tests/manual/test_pagination_backend.py`: 后端分页测试脚本（验证后端无问题）

- 🔧 **运维工具**

- 📝 **文档**

## [1.0.1] - 2026-02-03

### 修复
- ✅ **修复分页重复数据问题**
  - 问题：前端分页显示时，不同页面出现重复数据
  - 原因：每次查询都获取全部数据，在内存中分页，排序可能不稳定
  - 解决：实现数据库级别分页，使用 MongoDB 的 skip/limit
  - 效果：完全消除重复，性能提升 5 倍

### 优化
- ⚡ **数据库级别分页**
  - 使用 `collection.find().sort().skip().limit()` 在数据库层面分页
  - 只获取当前页需要的数据，避免内存溢出
  - 复合排序键 `[("logTime", 1), ("_id", 1)]` 确保排序稳定性

- 📊 **增强统计信息**
  - 添加中位数、标准差、变异系数
  - 添加分位数（p5, p25, p75, p95）
  - 添加趋势分析和异常检测
  - 添加时间分布分析（峰值时段、低谷时段）

- 🔍 **改进日志输出**
  - 添加详细的 MongoDB 查询日志

  - 添加 API 请求/响应日志

## [1.0.0] - 2026-01-15

### 新增
- 🎉 **初始版本发布**
- 🤖 **自然语言查询**
  - 支持中文自然语言查询设备数据
  - 基于 LangGraph 的多智能体系统
  - 支持多轮对话和上下文理解

- 🔍 **语义搜索**
  - 基于 FAISS 向量数据库
  - 使用 qwen3-embedding 模型
  - 支持 Rerank 重排序

- 📊 **数据可视化**
  - 表格展示查询结果
  - 统计信息展示
  - 支持数据导出

- 💬 **对话式交互**
  - 流式输出（SSE）
  - 实时步骤显示
  - 对话历史管理

- 📈 **对比分析**
  - 支持多设备数据对比
  - 自动识别对比查询
  - 生成对比分析报告

### 技术栈
- **后端**: FastAPI + LangGraph + MongoDB
- **前端**: HTML + TailwindCSS + JavaScript
- **AI**: Qwen3-32B (LLM) + qwen3-embedding + qwen3-reranker
- **数据库**: MongoDB 3.4+
- **向量库**: FAISS

### 核心功能
- 意图解析
- 语义映射
- 分片路由
- 并行数据获取
- 结果合成
- 缓存管理
- 上下文压缩
